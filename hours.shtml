This page shows how many core-hours1 were used by researchers each week, and which resource collection provided those hours. The following resource collections are available to campus researchers:

  • OSG - Open Science Grid (all OSG activity)
  • CAE - Computer Aided Engineering Department resources
  • CS - Computer Science Department resources
  • CHTC - Center for High Throughput Computing resources
  • GLOW - Grid Laboratory of Wisconsin resources

Hover over a legend entry to highlight the associated data in the graph. Hover over a graph element to see the date and value detail.

NOTES: 1) If a server with 16 cores is kept busy for 24 hours, it provides 16x24=384 core hours.
